# PALSAR-1 SAR Change Detection Tool
## 東日本大震災前後の地表変化可視化

## 概要
JAXAのPALSAR-1衛星データ（Tellus API経由）を使用し、
2011年東日本大震災前後の東北沿岸における地表変化を可視化するツールです。

## 使用データ
- データソース: Tellus（JAXA PALSAR-1）
- 震災前: 2011-03-03（東北沿岸）
- 震災後: 2011-04-08（東北沿岸）

## 使用技術
Python 3 / Tellus Traveler API / matplotlib / Pillow / NumPy / Google Colab

## 技術的課題と学び
- JPEGプレビュー画像は圧縮のためピクセル単位の精度に限界がある
- 撮影軌道・角度の違いにより差分にノイズが発生
- 精度向上には生データと位置合わせ処理が必要

## 背景・目的
QPS研究所が目指す準リアルタイム地球観測の社会的価値、
特に災害対応における衛星データの活用可能性を検証する目的で作成。
